Superposition

A quantum state can combine several allowed possibilities at once, with amplitudes that can reinforce or cancel each other.

Core idea

A useful beginner image is a musical chord: one sound made from multiple notes. Likewise, a superposition is one state built from several components. Measurement gives one allowed outcome, not the whole chord.

Visual intuition

State A + State B = One state

Superposition is a precise structure, not mere uncertainty.

Common mistake

It is not just “we do not know yet” in a classical sense.

Why it matters

It sets up interference and the probabilities seen in experiments.
